# Dygyn TMA import data pack

Created: 2026-05-06 17:23:06Z

Use order for import:
1. sources.csv
2. disciplines.csv
3. events.csv
4. participants_2026.csv
5. event_participants_2026.csv
6. results_2025_overall.csv
7. results_2025_by_discipline.csv
8. qualifier_2026_partial_results.csv

Notes:
- 2026 participants are based on public media sources as of May 2026.
- 2025 Dygyn.com table publishes discipline places/points, not full raw results for every athlete.
- Raw values are filled only where a public report provided them.
- Overall Dygyn scoring uses the lower sum of discipline places/points as better.
- Photo URLs point to source images from the YSIA participant profile article; verify copyright/permission before using them inside the public app.
