# Игры Дыгына — голосование

Telegram-приложение для болельщиков Игр Дыгына.

Здесь можно следить за участниками, сравнивать их результаты по видам, поддерживать фаворитов и видеть, как голосует всё сообщество.

## Идея

Это не сервис ставок. Это платформа голосования вокруг Игр Дыгына:

- посмотреть участников;
- изучить результаты спортсменов по видам;
- распределить очки поддержки;
- увидеть, за кого болеет сообщество;
- попасть в рейтинг болельщиков.

## Голосование

На одно событие у пользователя есть **100 очков уверенности**.

Их можно отдать:

- одному участнику — `100`;
- двум участникам — например `70 / 30` или `50 / 50`.

Так каждый голос имеет одинаковый вес, но пользователь сам решает: рискнуть и поддержать одного фаворита или распределить очки между несколькими участниками.

## Участники

У каждого участника должна быть понятная карточка:

- имя;
- улус / город;
- фото;
- краткое описание;
- поддержка болельщиков;
- результаты прошлых лет.

Главный акцент — таблица по 7 видам Игр Дыгына: бег, силовые дисциплины и другие результаты спортсмена по годам.

## Статистика

После голосов приложение показывает:

- процент поддержки каждого участника;
- сколько людей выбрали участника;
- сколько очков поддержки он получил;
- общий рейтинг болельщиков.

Игры Дыгына идут два дня, поэтому приложение должно показывать промежуточные итоги первого дня, второго дня и финальные результаты.

После завершения события очки рейтинга начисляются только за правильно выбранного победителя.

## Стиль

Визуальное направление: тёмный спортивный интерфейс с золотым акцентом.

Без казино-стиля, без букмекерских образов, без шума. Карточки, таблицы, рейтинг, чистая навигация.

## Главное правило

Игры Дыгына — голосование — про поддержку и интерес к спорту, не про деньги.

В проекте нет ставок, коэффициентов, депозитов, выводов, выплат, денежных призов или баланса пользователя.

Любые денежные механики возможны только после отдельного юридического решения.
