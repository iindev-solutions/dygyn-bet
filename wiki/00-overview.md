# 00. Обзор проекта

## Идея

Telegram Mini App для болельщиков Игр Дыгына: человек открывает приложение внутри Telegram, выбирает участника, за которого болеет, ставит виртуальные очки уверенности и видит общую статистику поддержки.

## Формат MVP

MVP не принимает деньги и не обещает выигрыш. В интерфейсе и API используются термины:

- прогноз;
- голос;
- фан-очки;
- очки уверенности.

Слова "ставка", "депозит", "вывод", "коэффициент", "выигрыш" в продуктовой логике лучше не использовать, пока нет юридически оформленной модели.

## Пользовательские сценарии

1. Пользователь открывает бота и нажимает кнопку Mini App.
2. Mini App отправляет на backend валидный `Telegram.WebApp.initData`.
3. Backend создаёт/обновляет пользователя по Telegram ID.
4. Пользователь выбирает событие и участника.
5. Backend сохраняет один прогноз на одного пользователя в рамках события.
6. Пользователь видит поддержку участников в процентах.
7. Администратор после завершения события вносит результаты.
8. Сервис начисляет фан-очки тем, кто угадал победителя.

## Не входит в MVP

- реальные денежные ставки;
- ввод/вывод денег;
- коэффициенты букмекера;
- KYC и проверка возраста;
- партнёрство с платёжными системами;
- автоматический парсинг официальных результатов;
- сложная админ-панель.
